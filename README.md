## Install Marp CLI locally

Install `@marp-team/marp-cli` in your Node projects.

```
$ yarn add --dev @marp-team/marp-cli
```

## Server mode (`--server` / `-s`)

Server mode (Pass directory to serve)

```
$ npx @marp-team/marp-cli -s ./slides
```
